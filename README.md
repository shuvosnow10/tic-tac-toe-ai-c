# Tic Tac Toe AI in C

A console-based Tic Tac Toe game written in C featuring:

- Two Player Mode
- Easy CPU
- Unbeatable Hard AI using Minimax
- Input validation
- Clean console interface

## Technologies
- C Programming Language
- Minimax Algorithm

## Compile

```bash
gcc main.c -o tictactoe
```

## Run

```bash
./tictactoe
```
